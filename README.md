# Provisioning a Windows Server on AWS using Terraform

This repository contains the implementation of the project 
"Provisioning a Windows Server on AWS using Terraform".

Included in this repo:
- **Code:** Terraform scripts (`.tf` files) for automating the creation of a Windows EC2 instance, VPC, security groups, and key pairs.
- **Docs:** Architecture diagrams and workflow charts explaining the infrastructure setup.
- **Screenshots:** Screenshots of deployed EC2 instances and AWS console verification.

The project demonstrates the use of **Infrastructure as Code (IaC)** to provision cloud resources efficiently and reproducibly. It showcases automation, secure configuration, and scalable deployment practices using Terraform on AWS.

## Skills & Tools Used
- Terraform
- AWS EC2
- VPC & Security Groups
- Infrastructure as Code (IaC)
